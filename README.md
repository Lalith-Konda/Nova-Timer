# Nova Timer

A modern, minimal speedcubing timer built from scratch using HTML, CSS and JavaScript.

## Features

### Completed
- ✅ Modular architecture
- ✅ Hold to start (300 ms)
- ✅ Spacebar support
- ✅ Touch support
- ✅ Ready-state indicator
- ✅ Reset
- ✅ +2
- ✅ DNF

### Planned
- ⏳ WCA Scramble Generator
- ⏳ Solve History
- ⏳ Statistics (Ao5, Ao12, Ao100)
- ⏳ Sessions
- ⏳ Settings
- ⏳ Themes
- ⏳ Backgrounds
- ⏳ Sounds & Vibration
- ⏳ Best Single / Average Alerts

## Tech Stack

- CSS3
- JavaScript (ES Modules)
